# NSQTech Fullstack Project

## Tech Stack

Frontend:
- Angular
- TypeScript
- Route Guards
- Role-based Access

Backend:
- Node.js
- Express
- REST APIs

## Features
- Login Authentication
- Role-based Dashboard
- Admin User Management
- Protected Routes
- API Integration

## How to Run

Frontend:
cd nsqtech-frontend
npm install
ng serve

Backend:
cd nsqtech-backend
npm install
node server.js
